# Semantic RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built with **Streamlit**, **FAISS**, **HuggingFace embeddings**, and **Google Gemini**.  
It allows users to upload documents, search them semantically, and generate accurate, context-aware answers using an LLM.

The project also demonstrates **RDF-like semantic triple extraction** and **similarity scoring** for educational purposes.
